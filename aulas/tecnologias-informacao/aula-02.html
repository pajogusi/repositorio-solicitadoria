Aula 02 - Hardware, software e sistemas operativos

Objetivos da Aula

  • Distinguir hardware e software.
  • Compreender a função de um sistema operativo.
  • Identificar componentes básicos de um computador.
  • Conhecer exemplos de software de aplicação.
  • Perceber a importância das atualizações e compatibilidade.
  • Relacionar estes conceitos com trabalho jurídico e administrativo.

Introdução

Para usar tecnologias de forma profissional, é necessário compreender os elementos básicos de um sistema informático.

Um computador, tablet ou telemóvel funciona através da relação entre hardware, software e sistema operativo.

Ideia principal:
Hardware é a parte física. Software é o conjunto de programas. O sistema operativo gere o funcionamento do equipamento e permite usar aplicações.

O que é hardware?

Hardware é o conjunto dos componentes físicos de um sistema informático.

É tudo aquilo que pode ser tocado: computador, monitor, teclado, rato, disco, impressora, processador, memória e outros dispositivos.

Exemplos de hardware:
Portátil, desktop, smartphone, impressora, scanner, disco externo, pen USB, router, webcam, monitor e teclado.

Componentes principais

Componente Função
Processador Executa instruções e processa tarefas
Memória RAM Guarda temporariamente dados em uso
Disco/SSD Armazena ficheiros e programas
Placa de rede Permite ligação à internet ou rede local
Monitor Apresenta informação visual

Memória RAM e armazenamento

Memória RAM e armazenamento não são a mesma coisa.

A RAM é usada enquanto o computador está a trabalhar. O armazenamento guarda ficheiros e programas de forma permanente.

Diferença essencial:
RAM ajuda o sistema a trabalhar no momento. Disco ou SSD guarda documentos, programas e dados mesmo depois de desligar o equipamento.

Periféricos

Periféricos são dispositivos ligados ao computador para entrada, saída ou armazenamento de informação.

  • Teclado.
  • Rato.
  • Impressora.
  • Scanner.
  • Webcam.
  • Microfone.
  • Disco externo.
  • Pen USB.

Em contexto jurídico, impressoras, scanners e leitores de cartão podem ser especialmente importantes.

O que é software?

Software é o conjunto de programas, aplicações e instruções que permitem ao hardware executar tarefas.

Sem software, o hardware não teria utilidade prática.

Exemplos de software:
Sistema operativo, processador de texto, browser, editor de PDF, folha de cálculo, programa de email, antivírus e aplicações de gestão documental.

Tipos de software

Tipo Função Exemplo
Software de sistema Permite o funcionamento geral do equipamento Sistema operativo
Software de aplicação Permite realizar tarefas concretas Editor de texto, browser, email
Software de segurança Ajuda a proteger o sistema Antivírus, firewall, gestor de passwords

Sistema operativo

O sistema operativo é o software principal de um computador ou dispositivo.

Gere recursos, ficheiros, programas, memória, periféricos, utilizadores, permissões e comunicação com o hardware.

  • Windows.
  • macOS.
  • Linux.
  • Android.
  • iOS.

Funções do sistema operativo

  • Permitir abrir e fechar programas.
  • Gerir ficheiros e pastas.
  • Controlar hardware e periféricos.
  • Gerir memória e processamento.
  • Controlar contas de utilizador.
  • Aplicar permissões e segurança.
  • Permitir ligação à internet.
  • Instalar atualizações.

Aplicações de produtividade

Aplicações de produtividade são programas usados para criar, editar, organizar e comunicar informação.

  • Processadores de texto.
  • Folhas de cálculo.
  • Programas de apresentação.
  • Leitores e editores de PDF.
  • Calendários digitais.
  • Clientes de email.
  • Gestores de tarefas.
  • Ferramentas de videoconferência.

Software jurídico e administrativo

Em contexto jurídico e administrativo, podem existir ferramentas específicas.

  • Gestão de processos.
  • Arquivo documental.
  • Assinatura digital.
  • Consulta de bases de dados jurídicas.
  • Submissão de atos em plataformas oficiais.
  • Emissão e organização de minutas.
  • Gestão de clientes e prazos.

Atualizações

Atualizações corrigem erros, melhoram segurança e podem acrescentar funcionalidades.

Ignorar atualizações durante muito tempo pode deixar o sistema vulnerável.

Ponto essencial:
Atualizações não são apenas estética ou novidade. Muitas vezes corrigem falhas de segurança.

Compatibilidade

Compatibilidade significa que equipamentos, sistemas e programas conseguem funcionar corretamente entre si.

Um documento pode abrir mal se for criado num formato incompatível ou numa versão diferente do software.

Exemplo:
Enviar um documento editável quando devia ser PDF pode alterar formatação, margens, paginação ou conteúdo visível.

Licenças de software

O software pode ter diferentes modelos de licença.

  • Software comercial pago.
  • Software gratuito.
  • Software livre ou open source.
  • Subscrições mensais ou anuais.
  • Licenças empresariais.

Usar software sem licença válida pode gerar problemas legais, profissionais e de segurança.

Segurança básica do equipamento

Um equipamento profissional deve ser protegido.

  • Password forte.
  • Bloqueio automático de ecrã.
  • Atualizações regulares.
  • Antivírus ou proteção equivalente.
  • Backup.
  • Separação entre uso pessoal e profissional.
  • Cuidado com pens USB desconhecidas.

Dispositivos móveis

Telemóveis e tablets também são sistemas informáticos.

Podem conter emails, documentos, fotografias, mensagens, dados de clientes, aplicações bancárias e autenticação digital.

Devem ser protegidos com PIN, biometria, bloqueio de ecrã e cuidado com aplicações instaladas.

Ligação à Solicitadoria

Na Solicitadoria, hardware e software são instrumentos de trabalho.

Um computador lento, inseguro ou desorganizado pode provocar atrasos, perda de documentos, falhas de comunicação e riscos de confidencialidade.

Ligação prática:
A competência digital começa por saber usar bem o equipamento, proteger ficheiros, manter software atualizado e escolher ferramentas adequadas.

Erros Frequentes

  • Confundir hardware com software.
  • Não perceber a função do sistema operativo.
  • Ignorar atualizações.
  • Guardar documentos apenas num dispositivo sem backup.
  • Usar software sem licença ou de origem duvidosa.
  • Trabalhar com dados profissionais em equipamentos desprotegidos.

  4. Qual é a diferença entre RAM e armazenamento?
  5. Porque as atualizações são importantes?
  6. Indica três exemplos de software usado em contexto profissional.

Respostas Comentadas

  1. É o conjunto dos componentes físicos de um sistema informático.
  2. É o conjunto de programas e instruções que permitem executar tarefas.
  3. Gerir recursos, ficheiros, aplicações, hardware, utilizadores e segurança do sistema.
  4. RAM é memória temporária em uso; armazenamento guarda ficheiros e programas de forma permanente.
  5. Porque corrigem erros, melhoram segurança e mantêm o sistema funcional.
  6. Processador de texto, folha de cálculo, browser, email, editor de PDF ou software de gestão documental.

Resumo em 5 Pontos

  1. Hardware é a parte física do sistema informático.
  2. Software é o conjunto de programas e aplicações.
  3. O sistema operativo gere o funcionamento geral do equipamento.
  4. Atualizações, licenças e compatibilidade são importantes para segurança e produtividade.
  5. Na Solicitadoria, equipamento e software mal geridos podem criar riscos profissionais e jurídicos.
