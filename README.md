# Universidade da Vida - Direito e Solicitadoria Independente

Projeto pessoal de estudo baseado na Licenciatura em Solicitadoria.

Objetivos:
- Aprender Direito
- Organizar conhecimento
- Construir uma plataforma própria de estudo

Regras:
- Primeiro página principal
- Depois primeira cadeira
- Depois Aula 01
- Só depois quizzes e progressão

Este projeto não substitui formação oficial nem aconselhamento jurídico.
